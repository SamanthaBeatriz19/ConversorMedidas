# ConversorMedidas

Este projeto consiste no desenvolvimento de um conversor de unidades de medida em Java, com interface<br>
gráfica (Swing). O programa deve ser capaz de carregar dinamicamente as classes de conversão, de modo a <br>
aceitar até mesmo aquelas que sejam criadas após a sua compilação.

O programa, ao partir, deve automaticamente vasculhar um diretório pré-determinado e carregar as classes <br>
conversoras. Novas classes podem ser adicionadas a qualquer tempo e as existentes podem ser removidas <br>
também livremente. Todos os possíveis erros destas operações devem ser devidamente tratados pelo sistema <br>
de exceções

na faculdade de tecnologia da UNICAMP.
